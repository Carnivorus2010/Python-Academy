# Course Plan

## Module 1 — Python Refresh
Variables, strings, lists, dictionaries, loops. Build a token inspector.

## Module 2 — Functions and Decisions
Functions, return values, validation, conditionals. Build a hex-color validator.

## Module 3 — Files and Paths
`pathlib`, JSON, file I/O, deterministic output. Generate a config file.

## Module 4 — Errors and CLI Programs
Exceptions, `argparse`, exit codes, helpful error messages.

## Module 5 — Linux Automation
`subprocess.run`, safe command execution, dry-run behavior, service checks.

## Module 6 — Testing and Refactoring
pytest, pure functions, fixtures, separating logic from I/O.

## Module 7 — Theme Generator
Generate Foot, Yazi, and Neovim configuration from one token source.

## Module 8 — Release Helper
Check Git state, update versions, create commits and tags, support `--dry-run`.
